📖 About Project Management

What is This App?

This is a simple project management dashboard that helps you organize and track your projects. Add projects, view them in a beautiful card layout, and delete them when completed.

🎯 Features

  • Create Projects: Add new projects by entering a name and pressing Enter or clicking the button
  • View Projects: See all your projects displayed in an organized grid
  • Delete Projects: Remove projects you no longer need with a confirmation
  • Persistent Storage: Your projects are saved automatically in your browser
  • Responsive Design: Works perfectly on desktop, tablet, and mobile devices

📱 How to Use

  1. Go to the Home page
  2. Type your project name in the input field
  3. Press Enter or click the Add Project button
  4. Your project appears instantly in the project list
  5. Click Delete to remove a project
  6. All your projects are automatically saved!

🛠️ Technology Stack

  • HTML: Page structure and content
  • CSS: Beautiful styling with gradients and responsive layout
  • JavaScript: Interactive functionality and data management
  • LocalStorage: Data persistence in the browser

💾 Data Storage

This app uses your browser's LocalStorage to save projects. Your data is stored locally on your device and persists even after you close the browser. No data is sent to any server.

📝 Project Information

Each project card shows:

  • Project name
  • Date created
  • Delete button

🎨 Design

The app features a modern gradient design with smooth animations. Hover over elements to see interactive effects. The layout adapts beautifully to all screen sizes.
